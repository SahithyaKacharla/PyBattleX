# ⚔️ PyBattleX

PyBattleX is an interactive Python terminal game featuring a turn-based battle system. Players can select unique heroes, attack enemies, use special abilities, and heal strategically to defeat AI opponents.

Built using Python and random-based mechanics, the game delivers engaging and unpredictable combat experiences while demonstrating core programming concepts.

---

## 🎮 Features

* Hero Selection System
* Turn-Based Battle Mechanics
* Attack and Defense Actions
* Special Abilities
* Healing System
* AI Opponent Battles
* Randomized Combat Outcomes
* Interactive Terminal Interface

---

## 🛠️ Tech Stack

* Python

---

## 🚀 How to Run

Clone the repository:

```bash
git clone https://github.com/SahithyaKacharla/PyBattleX.git
```

Navigate to the project directory:

```bash
cd PyBattleX
```

Run the game:

```bash
python battle_game.py
```
